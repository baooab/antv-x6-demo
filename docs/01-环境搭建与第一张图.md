# 第一部分：环境搭建与第一张图

## 1. 目标

读完本部分，你应能：

- 理解本教程项目的技术选型与目录结构；
- 在 React 中用容器挂载一个 X6 `Graph`；
- 添加矩形节点与连线，并在组件卸载时正确销毁画布；
- 理解示例里与交互相关的几项画布配置。

## 2. 技术栈

| 项 | 说明 |
| --- | --- |
| [Vite](https://vite.dev/) | 开发与构建工具，冷启动快 |
| [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) | UI 与类型 |
| [@antv/x6](https://www.npmjs.com/package/@antv/x6) | 图编辑与可视化引擎 |
| [@antv/x6-react-shape](https://www.npmjs.com/package/@antv/x6-react-shape) | 从第二部分起在示例中注册 React 节点；第一部分正文仍以内置矩形为主 |

## 3. 依赖说明

除 `react`、`react-dom` 外，与 X6 直接相关的依赖包括：

- **`@antv/x6`**：核心库。
- **`tslib`**：X6 构建产物会引用；使用 Vite 打包时需在项目中安装，否则生产构建可能报无法解析 `tslib`。

安装示例：

```bash
npm install @antv/x6 @antv/x6-react-shape tslib
```

## 4. 与本部分相关的目录

```
src/
  App.tsx              # 页面布局与标题
  App.css              # 全屏布局、画布容器占位
  index.css            # 全局样式
  components/
    DemoGraph.tsx      # 本部分核心：创建 Graph、示例节点与边
```

要点：**画布容器必须有确定高度**。示例通过 `App.css` 中 `.app` 使用 `height: 100%` 与 `flex: 1` 的 `.app-main`，使内部 `.demo-graph-canvas` 以绝对定位铺满剩余区域；再配合 `Graph` 的 `autoResize: true`，画布会随窗口变化调整尺寸。

## 5. 在 React 中挂载 Graph

推荐模式：`useRef` 持有 DOM 容器，`useEffect` 内创建 `Graph`，在 effect 的清理函数中调用 `graph.dispose()`。

原因简述：

- 创建 `Graph` 属于副作用，不应在渲染阶段执行；
- React 18+ 严格模式下开发环境会对 effect 执行两次挂载/卸载，**必须** `dispose`，否则会重复绑定或泄漏；
- 路由切换离开页面时同样需要释放资源。

核心代码位于 `src/components/DemoGraph.tsx`。

## 6. 第一张图：节点与边

X6 中常用流程：

1. `graph.addNode({ x, y, width, height, label, ... })` 返回节点实例；
2. `graph.addEdge({ source, target, attrs, ... })` 中 `source` / `target` 可传入上一步的节点对象。

`attrs` 用于设置边路径样式，例如描边颜色、线宽、箭头 `targetMarker` 等，与 SVG 属性体系一致。

## 7. 本示例中的画布配置（摘抄说明）

以下与 `DemoGraph.tsx` 中注释一致，便于对照源码。

| 配置 | 作用 |
| --- | --- |
| `autoResize: true` | 随容器尺寸变化更新画布宽高 |
| `grid: true` | 显示默认网格，便于对齐（可改为对象做细调） |
| `panning` | 是否允许拖动画布空白处**平移整个视口**；与下面「节点移动」是不同能力 |
| `translating.restrict: true` | 拖动**节点**时，限制在画布可视区域（内部基于 `getGraphArea()`） |
| `mousewheel.enabled` + `modifiers` | 启用滚轮缩放；要求按住 **Ctrl**（Windows/Linux）或 **⌘**（macOS）再滚轮，减少误触 |

若将 `panning` 设为 `true`，页头文案需同步说明「拖空白平移」；当前仓库示例默认关闭平移，仅作节点拖拽与缩放演示。
