# 第 07 课 · 落地替换：从模具预览到 React 节点

**交互 Demo**：`src/tutorial/demos/Lesson07DropReplace.tsx` · 页面哈希 `#l07`

## 目标

从左侧模具拖入的节点带有 **`data.type`**（如 `llm`、`kb`），形状为 **`agent-stencil-card`**。落地后应换成 **`createAgentCard`** 生成的 **`agent-react-card`**，并保持 **坐标不变**。

## 实现思路

监听 **`node:added`**：

1. 若 **`node.shape === 'agent-react-card'`**，说明已是最终形态，忽略；
2. 若存在 **`data.type`**，用该 key 从配置表里取 **`AgentCardConfig`**；
3. 读取当前 **`node.position()`**，**移除**旧节点，**添加**新节点并 **`position(x, y)`**。

配置来源：本仓库为 **`public/data/agent-flow.json`**（可改为接口请求）。**`ensureAgentFlowShapesRegistered`** 需已注册 **React 卡** 与 **模具卡** 等全部形状。

## 本课 Demo 特点

- 初始 **主画布为空**，请从左侧 **拖入** 节点观察替换；
- 模具分组与第 08 课完整版相比做了精简，仅用于演示机制。
