# 第 01 课 · 最简画布：Graph 与内置节点

**交互 Demo**：`src/tutorial/demos/Lesson01MinimalGraph.tsx` · 页面哈希 `#l01`

## 目标

- 用 `new Graph({ container })` 把 X6 挂到 DOM 上；
- 添加一个内置 **`rect`** 节点，确认渲染与 **Ctrl / ⌘ + 滚轮** 缩放（本仓库视口约定）；
- 在组件卸载时 **`graph.dispose()`**。

## 步骤说明

1. 准备一个 `div` 作为 **`container`**（本仓库用 `useRef` 获取）。
2. 实例化（与 Demo 一致，使用视口预设展开）：

   ```ts
   import { agentFlowViewport } from '../../agent-flow/graphOptions'

   const graph: Graph = new Graph({
     container: el,
     autoResize: true,
     grid: true,
     ...agentFlowViewport,
   })
   ```

   **本仓库约定（`src/agent-flow/graphOptions.ts`）**：**`agentFlowViewport`** 内含 **`agentFlowMousewheel`**，并统一主画布行为——**关闭空白处拖拽平移**（`panning: false`；X6 默认否则可左键拖动画布）、**节点限制在画布内**（`translating.restrict`）、**滚轮缩放需按住 Ctrl（Windows）或 ⌘（macOS）**（`modifiers: ['ctrl', 'meta']`）。左侧 Stencil 小图仍可在 `stencilGraphOptions` 中单独配置平移，与主画布无关。

3. 使用 **`graph.addNode`** 增加内置矩形，通过 `attrs.body`、`label` 控制圆角与文字（见 Demo 源码）。

4. 在 `useEffect` 的 **return** 中调用 **`graph.dispose()`**。
