# 第 08 课 · 完整应用：智能体流程编排

**交互 Demo**：`src/tutorial/demos/Lesson08FullApp.tsx`（内部渲染 **`AgentFlowView`**）· 页面哈希 `#l08`

## 目标

把前序课程的能力 **拼成与官方「智能体流程编排」一致的演示**：

- **`Snapline`** + 主画布 **`connecting`** + **`bindAgentFlowInteractions`**；
- 左侧 **Stencil** 多分组、**`stencil.load`** 填充业务与数据类节点；
- **`fetch('/data/agent-flow.json')`** 加载类型配置；
- **`node:added`** 模具落地替换；
- 画布上预置 **开始 / 大模型 / 结束** 节点及两条示例边（**`agent-edge`**）。

## 入口与模块

| 模块 | 职责 |
| --- | --- |
| **`src/agent-flow/AgentFlowView.tsx`** | 组装 Graph、Stencil、事件、异步加载与初始连线 |
| **`graphOptions.ts`** | 滚轮、`createAgentFlowConnecting`、`highlighting` |
| **`registerShapes.ts`** | 边、React 卡、Flow 起止、模具预览节点（`ensureAgentFlowShapesRegistered`） |
| **`factory.ts`** | `createAgentCard`、`createFlowCard`、`createAgentStencilCard` |
| **`graphInteractions.ts`** | 端口与边交互 |
| **`public/data/agent-flow.json`** | 各 `key` 对应卡片文案与主题 |

## 与官方示例的关系

交互与视觉对齐 AntV 展示案例：[智能体流程编排](https://x6.antv.antgroup.com/examples/showcase/practices/#agentFlow)。本仓库将逻辑拆到上述文件，并用 **教程页** 支持从第 01 课逐步调试到本课。

## 学完本课后

可自行扩展：History 撤销、MiniMap、导出图片、将 **`node.data`** 同步到后端保存流程 JSON 等。
