# 第 02 课 · 自定义边与四向连接桩

**交互 Demo**：`src/tutorial/demos/Lesson02CustomEdge.tsx` · 页面哈希 `#l02`

## 目标

- 用 **`Graph.registerEdge`** 注册名为 `agent-edge` 的边样式；
- 在节点上配置 **`ports`**（本仓库为四向圆点，见 `src/agent-flow/ports.ts` 的 **`fourWayPorts`**）；
- 配置 **`connecting`**：平滑连接器、磁吸、`createEdge` 使用注册的边形状；
- 理解 **`createAgentFlowConnecting(() => graph)`** 必须用箭头函数懒取 `graph`，避免 `const graph = new Graph({ connecting: … })` 在求参阶段引用尚未完成初始化的变量。

## 关键点

### 1. 注册边

`src/agent-flow/registerShapes.ts` 中的 **`registerAgentEdgeShape`**：继承内置 `edge`，在 `attrs.line` 上设置描边、箭头 `targetMarker` 等。

### 2. 连接桩

`fourWayPorts` 为每组定义 **`magnet: true`**，圆点才能作为连线起点/终点；默认用 **`style.visibility: hidden`** 隐藏，后续课程再讲悬停显示。

### 3. connecting 配置

统一集中在 **`src/agent-flow/graphOptions.ts`** 的 **`createAgentFlowConnecting`**，包含：

- `connector: { name: 'smooth' }`、`connectionPoint: 'anchor'`
- `allowBlank: false`、`snap`、`allowLoop: false` 等策略
- **`createEdge`**：返回 `getGraph().createEdge({ shape: 'agent-edge' })`
- **`validateConnection`**：本示例要求必须落到目标 **magnet**（`!!targetMagnet`）

### 4. 演示节点

本课仍使用内置 **`rect`**，仅挂上 `ports: fourWayPorts`，放置两个节点后从一侧圆点拖线到另一侧即可。
