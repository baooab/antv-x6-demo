## 2. 依赖说明

与 X6 直接相关的依赖通常包括：

- **`@antv/x6`**：核心库。
- **`@antv/x6-react-shape`**：注册 React 形状、挂载视图。
- **`tslib`**：部分 X6 构建产物会引用；若构建报无法解析 `tslib`，请在项目中安装。

安装示例：

```bash
npm install @antv/x6 @antv/x6-react-shape tslib
```

## 3. 在 React 中创建与销毁 Graph

推荐模式：`useRef` 指向画布容器 DOM，`useEffect` 内 `new Graph({ container })`，在 **effect 的清理函数** 中调用 **`graph.dispose()`**。

原因简述：

- 创建 `Graph` 属于副作用，不应在渲染阶段执行；
- React 18+ 开发严格模式可能对 effect **重复挂载/卸载**，若不做 `dispose` 会泄漏或产生重复实例；
- 切换教程课程时，界面会用 `key` 卸载旧 Demo，同样依赖各 Demo 内的清理逻辑。

模具组件 **`Stencil`** 在卸载时也应调用 **`stencil.dispose()`**（见第 06、07、08 课对应源码）。

## 4. 画布容器尺寸

**画布容器必须有可计算的高度**（或父级链路上有明确高度），否则 `Graph` 可能高度为 0。本仓库通过 `.app`、`.app-main`、`.tutorial-demo-wrap` 等样式让演示区域在视口内占满剩余空间；各课 Demo 内对画布根节点使用 `position: absolute; inset: 0` 或 flex 子项 `flex: 1` 配合 **`autoResize: true`**，使画布随容器变化。
